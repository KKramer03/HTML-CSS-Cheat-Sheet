HTML Cheatsheet!

The most important parts of an HTML page

The most important parts of a webpage are <!Doctype HTML> and <html>. These are declarations that tell the computer that you are using HTML. The other important tags to start a page are <head>, <title>, and <body>. The <head> tag encompasses the <title> tag and the link to an external CSS file. Any text in the <title> tag is displayed as text in the tab menu, while the <body> tag ecompasses all content displayed on the website itself. <!Doctype HTML> is an empty element and requires no closing tag.

Links

There are several types of links, all using <a href=""> tags. The most common links are to external sites using either words or images. Other types of <a href=""> tags are ones that link to other locations on the same site or locations on the same page. To link to another page on the same site, input the pages filename "example.html" into the href quotations. To link to a location on the same page, the element you want to link to must have an id="something" in it's opening tag. Then #something must be put in the a href quotations. You can open a new tab by adding target="_blank" in the opening <a href=""> tag.
